SIO2Arduino is an open-source implementation of the Atari SIO device protocol that runs
on Arduino hardware. Arduino hardware is open-source and can be assembled by hand or
purchased preassembled.

For more information on SIO2Arduino, see the website at: 
http://www.whizzosoftware.com/sio2arduino

For more information on the Arduino platform, go to:
http://www.arduino.cc
